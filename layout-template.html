    M02 Code
    <div class="container mt-3 px-3"> <div class="row"> <div class="col"> <div class="accordion"> <div class="card"> <div class="card-header bg-white px-2 position-relative"> <h2 class="mb-0"> <button type="button" id="collapse-toggle-m02" data-target="#c-panel-m02" aria-controls="c-panel-m02" aria-expanded="true" data-toggle="collapse" class="btn btn-link btn-block text-left text-body collapse-chevron" > Lorem </button> </h2> </div> <div id="c-panel-m02" aria-labelledby="collapse-toggle-m02" class="collapse show" > <div class="card-body"> <p>Lorem ipsum.</p> </div> </div> </div> </div> </div> </div> </div>
    Important
    • Button id, data-target, aria-expanded, aria-controls must be set properly.
      • id refers to the button. Must be unique per page. No duplicate ID is allowed.
      • data-target refers to the target collapse div. In the example, it is the #c-panel-m02. Must use hash (#) symbol. This is used for the purpose for javascript.
      • aria-controls is quite similar with data-target, except aria-controls doesn't use #. This is used for the purpose of accessibility.
      • If target div (in the example, it is the div with id="c-panel-m02") is using class show (panel is expanded by default), then aria-expanded must be set to true.
    • Target div (in the example, it is the div with id="c-panel-m02") must have unique ID. The ID will be referenced by the toggle button.
    • For accessibility purpose, target div requires aria-labelledby attribute with value equal to the id of the toggle button. In the example, the value is aria-labelledby="collapse-toggle-m02".
    • To set the target div to be opened by default, add class of show to the target div. Do not forget to set aria-expanded on button element to true
